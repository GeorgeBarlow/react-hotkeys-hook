# useIsHotkeyPressed

The `useIsHotkeyPressed` hook just returns the `hotkeys.isPressed` function and works exactly the same.

```ts
const isPressed = useIsHotkeyPressed();

isPressed('return'); // Returns true if Return key is pressed down.
```
